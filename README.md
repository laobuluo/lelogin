# LeLogin - WordPress 自定义登录界面美化插件

一款强大的WordPress登录界面自定义美化插件，允许您完全自定义登录页面的外观和样式。

## 功能特性

### 🎨 视觉自定义
- **背景设置**：自定义背景颜色或上传背景图片
- **Logo定制**：上传自定义Logo，设置Logo宽度和链接
- **颜色方案**：完全自定义按钮、链接、文本颜色
- **表单样式**：自定义表单背景、圆角、内边距

### ⚙️ 高级功能
- **自定义CSS**：支持添加自定义CSS代码进行深度定制
- **实时预览**：设置页面提供预览链接
- **响应式设计**：自动适配移动设备
- **平滑动画**：内置过渡动画效果

## 安装方法

### 方法一：手动安装
1. 下载插件文件
2. 将整个 `lelogin` 文件夹上传到 `/wp-content/plugins/` 目录
3. 在WordPress后台的"插件"页面激活插件

### 方法二：通过WordPress上传
1. 将插件文件夹压缩为 `lelogin.zip`
2. 在WordPress后台进入"插件" > "安装插件" > "上传插件"
3. 选择zip文件并安装
4. 激活插件

## 使用方法

### 基本设置
1. 激活插件后，进入 **设置** > **LeLogin**
2. 在设置页面中，您可以：
   - **常规设置**：设置背景颜色或背景图片
   - **Logo设置**：上传自定义Logo并配置相关选项
   - **颜色设置**：自定义按钮、链接、文本颜色
   - **表单样式**：调整登录表单的外观
   - **高级设置**：添加自定义CSS代码

### 设置说明

#### 背景设置
- **背景颜色**：选择登录页面的背景颜色
- **启用背景图片**：勾选后可以使用背景图片替代背景颜色
- **背景图片**：上传或输入背景图片URL

#### Logo设置
- **启用自定义Logo**：启用后使用自定义Logo替代WordPress默认Logo
- **Logo图片**：上传或输入Logo图片URL
- **Logo宽度**：设置Logo的显示宽度（50-500像素）
- **Logo链接地址**：点击Logo时跳转的URL
- **Logo标题**：Logo的alt和title属性

#### 颜色设置
- **按钮颜色**：登录按钮的背景颜色
- **按钮悬停颜色**：鼠标悬停时的按钮颜色
- **按钮文字颜色**：按钮上的文字颜色
- **链接颜色**：登录页面上链接的颜色
- **链接悬停颜色**：链接悬停时的颜色
- **文本颜色**：表单标签和文本的颜色

#### 表单样式
- **表单背景颜色**：登录表单的背景颜色
- **表单圆角**：表单边框的圆角大小（如：4px, 8px）
- **表单内边距**：表单内部的内边距（如：30px）

#### 高级设置
- **自定义CSS**：添加自定义CSS代码来进一步美化登录页面
